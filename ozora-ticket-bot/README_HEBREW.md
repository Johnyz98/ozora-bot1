# בוט כרטיסים לאוזורה — מדריך הקמה

הבוט הזה בודק בכל חצי שעה את דף "Ticket Find ForSale" של אוזורה,
קורא כמה הנחה יש כרגע, וכשההנחה מגיעה לסף שתבחר (ברירת מחדל: 90 יורו) —
שולח לך הודעה לטלפון עם המחיר, מספר הכרטיס ולינק ישיר לקנייה.

הבוט רץ בענן בחינם (GitHub Actions), כך שהוא עובד גם כשהמחשב שלך כבוי.

---

## מה תצטרך (חד פעמי, בערך 15 דקות)

1. חשבון GitHub (חינם) — כאן הבוט "גר" ורץ.
2. דרך לקבל הודעות. שתי אפשרויות:
   - **טלגרם** — הכי קל, 3 דקות, מומלץ. ⭐
   - **וואטסאפ דרך Twilio** — עובד, אבל ההקמה ארוכה יותר.

> המלצה ממני: גם אם רצית וואטסאפ, טלגרם פשוט הרבה יותר קל להקים
> ועושה בדיוק את אותו דבר — צ'פצ'וף בטלפון תוך שניות. הקוד תומך בשניהם,
> אז תבחר מה שנוח. אפשר גם להתחיל מטלגרם ולהוסיף וואטסאפ אחר כך.

---

## שלב 1 — להעלות את הקוד ל-GitHub

1. היכנס ל-https://github.com והירשם (אם אין לך חשבון).
2. למעלה מימין לחץ על ➕ ואז **New repository**.
3. תן שם, למשל `ozora-bot`. אפשר להשאיר אותו **Private**. לחץ **Create repository**.
4. במסך שנפתח לחץ על הקישור **uploading an existing file**.
5. גרור לתוך החלון את כל הקבצים מהתיקייה שקיבלת:
   `monitor.py`, `requirements.txt`, ואת התיקייה `.github`.
   (אם הגרירה של תיקיית `.github` לא עובדת — ראה הערה בתחתית המדריך.)
6. למטה לחץ **Commit changes**.

הקוד עכשיו בענן. עוד לא רץ — קודם נחבר את ההתראות.

---

## שלב 2א — טלגרם (הדרך הקלה) ⭐

1. פתח טלגרם, חפש למעלה **BotFather** (עם וי כחול), פתח שיחה.
2. שלח לו: `/newbot` ועקוב אחרי ההוראות (שם לבוט + שם משתמש שמסתיים ב-`bot`).
3. בסוף הוא ייתן לך **טוקן** — שורה כמו `7712345678:AAH...`. שמור אותה.
4. עכשיו צריך את ה-**chat id** שלך:
   - חפש בטלגרם בוט בשם **userinfobot**, פתח שיחה ושלח לו כל הודעה.
   - הוא יחזיר לך מספר בשורת `Id` — זה ה-chat id שלך.
5. **חשוב:** פתח את הבוט החדש שיצרת (בלינק ש-BotFather נתן) ולחץ **Start** /
   שלח לו "שלום" — אחרת הוא לא יוכל לשלוח לך הודעות.

עכשיו דלג לשלב 3.

---

## שלב 2ב — וואטסאפ דרך Twilio (אופציונלי)

1. הירשם ב-https://www.twilio.com/try-twilio (חינם, יבקש אימות טלפון).
2. בקונסול, בתפריט: **Messaging → Try it out → Send a WhatsApp message**.
3. תקבל מספר sandbox והוראה לשלוח ממנו הודעת `join xxxx-xxxx` מהוואטסאפ שלך —
   עשה זאת כדי לחבר את המספר שלך.
4. בדף הראשי של הקונסול העתק: **Account SID** ו-**Auth Token**.
5. ה-`from` הוא מספר ה-sandbox, בדרך כלל: `whatsapp:+14155238886`
   וה-`to` הוא המספר שלך, למשל: `whatsapp:+9725XXXXXXXX`

---

## שלב 3 — לחבר את הפרטים לבוט (Secrets)

1. ב-GitHub, ברפו שלך, לחץ למעלה על **Settings**.
2. בצד שמאל: **Secrets and variables → Actions**.
3. לחץ **New repository secret** והוסף את אלה (שם בדיוק כמו שכתוב, ערך לפי שלך):

   אם בחרת טלגרם:
   - `TELEGRAM_BOT_TOKEN` → הטוקן מ-BotFather
   - `TELEGRAM_CHAT_ID` → ה-id מ-userinfobot

   אם בחרת וואטסאפ:
   - `TWILIO_ACCOUNT_SID`
   - `TWILIO_AUTH_TOKEN`
   - `TWILIO_FROM`  (למשל `whatsapp:+14155238886`)
   - `TWILIO_TO`    (המספר שלך, למשל `whatsapp:+9725XXXXXXXX`)

   רשות (לשניהם):
   - `EUR_THRESHOLD` → הסף שמפעיל התראה. אם לא תוסיף, ברירת המחדל היא `90`.
     רוצה גם 90 וגם 100? תשים `90` והבוט יתריע על כל הנחה של 90 ומעלה.

---

## שלב 4 — להפעיל ולבדוק

1. ב-GitHub לחץ למעלה על **Actions**. אם יש כפתור ירוק להפעלת workflows — לחץ עליו.
2. בחר את **Ozora ticket check** מצד שמאל, ואז **Run workflow** (כפתור מימין) → **Run workflow**.
3. אחרי דקה לחץ על הריצה כדי לראות את הלוג. תראה שורה כמו
   `Current discount: 60 EUR ...` — סימן שהבוט קורא את הדף.
4. רוצה לוודא שההתראה עצמה עובדת? היכנס ל-Secrets, שנה זמנית את `EUR_THRESHOLD` ל-`10`,
   הפעל ידנית שוב — אמורה להגיע לך הודעה. אחר כך תחזיר את הסף ל-`90`.

מכאן הבוט ירוץ אוטומטית כל חצי שעה. כשתגיע הנחה של 90+ תקבל הודעה עם לינק,
תפתח, תכניס מייל, תלחץ APPROVE — ותאשר במייל תוך שעה.

---

## אחרי שקנית

כדי שהבוט יפסיק להתריע: **Actions → Ozora ticket check → ... (שלוש נקודות) → Disable workflow**.
לחזור בעתיד? אותו מקום, **Enable**.

---

## הערות קטנות

- GitHub עוצר workflows מתוזמנים אם אין שום פעילות ברפו במשך 60 יום.
  אם הבוט רץ הרבה זמן והפסיק — פשוט תיכנס ל-Actions ותריץ ידנית פעם אחת, וזה מתחדש.
- הזמן ב-cron הוא לפי UTC, וגיטהאב לפעמים מאחר בכמה דקות בריצות מתוזמנות. זה תקין.
- אם גרירת התיקייה `.github` לא עבדה בשלב 1: ברפו לחץ **Add file → Create new file**,
  בשם הקובץ כתוב בדיוק `.github/workflows/check.yml`, הדבק לתוכו את תוכן הקובץ
  `check.yml` שקיבלת, ולחץ Commit.
- הבוט רק מתריע ושולח לך לינק — את הקנייה והכנסת המייל אתה עושה בעצמך,
  כדי שלא תתחייב בטעות לכרטיס שלא רצית.
